如何架設個人網站

  • DURATION

    06 Month

  • WEEKLY

    03 hours

  • FEE

    From: $699

Apply now

前言

相信很多人都想透過部落格的方式將知識寫成一篇篇的文章方便瀏覽，其中不外乎是使用Medium、Wix、Blogger、Wordpress等來架設網站或設計版面，而我自己是使用Medium來撰寫文章，但使用到後期發現無法滿足個人高自由度的需求XD!

.

首先是版面配置，Medium在更新之後就無法免費調整文章版面配置，結果我的Medium看起來就像是新聞網站完全無美感而言!
不放棄的我再度尋找新的架站機會，所幸在今年在Hahow上發現這堂《 Github 免費架站術！輕鬆打造個人品牌 》

首先我點選學員作業發現網站的完成度很高，再加上版面配置自由度非常高，我就決定手刀買這堂CP值超高的課程囉!!

目錄

《一》設定網站版面

《二》本地端運行網站

《三》客制化網站

《四》部署線上網頁

《五》使用Google功能

《一》設定網站版面

1.進入Hugo官網 ➨ 點選Themes ➨ 選擇網站版面 ➨ 下載zip檔

2.主檔案中的exampleSite拉至桌面 ➨ 新增themes資料夾 ➨ 將主檔名改成*name*-hugo ➨ 移至themes

3.下載hugo.exe至exampleSite

《二》本地端運行網站

1.cmd exampleSite ➨ 輸入hugo server ➨ 複製網址(通常為http://localhost:1313/)

《三》客制化網站

exampleSite ----- content ----- blog # 文章內容 md
                             |
                             |- data ----- homepage.yml # 首頁設計 yml
                             |
                             |- static ----- images # 圖片
                             |
                             |- themes ----- assets -----scss # 主題色 & 字體
                             |
                             |- config.toml # 個人資料
1.網址及網站icon設定
  • baseURL(網址)為https://*Username*.github.io
  • title可自由命名
  • 將選定的icon命名為favicon.png(網站圖示)

2.字體程式碼設定
  • google fonts選擇字體 ➨ vscode搜尋.content { ➨ 複製程式碼貼上
@import url('https://fonts.googleapis.com/css2?family=PT+Serif&display=swap');
.content {
  * {
    font-family: 'PT Serif', serif;
    margin-bottom: 20px;
  }

  • hugo markup選擇程式碼模板 ➨ Highlight內的Short snippets ➨ 複製程式碼至config.toml ➨ 修改style

[markup]
[markup.highlight]
style = 'friendly'
tabWidth = 4
3.設定文章版面
  • vscode搜尋.Summary ➨ 修改為Description
<p class="card-text mb-4">{{ .Description | truncate 100 }}</p>

《四》部署線上網頁

1.新增repository ➨ Repository name 必須為帳號名稱.github.io

2.透過GitHub Desktop下載專案(位置:本機/文件/GitHub/)

3.上傳程式碼至GitHub
  • cmd exampleSite ➨ 輸入hugo ➨ 複製exampleSite/public內的檔案 ➨ 貼至本機/文件/GitHub/Username.github.io
  • GitHub Desktop Commit to main ➨ Fetch origin

《五》使用Google功能

1.設定Google Analytics

  • 將追蹤ID貼至config.toml中的google_analitycs_id

2.設定Google Search Console

  • 貼上網址要求建立索引

結論

相信大家在架站的過程中絕對可以體會到使用GitHub Pages的便利性及高自由度，整個過程絕對比透過worldpress架站還來得有效率，我操作幾次下來幾乎都可以在半小時內架好一個完整的模板，一切都設定好後大家就可以將重心放在文章的質與量囉!

.

而Hahow課程中其實還有很多強大的功能和markdown寫法我這邊就不一一介紹了，大家有興趣的話可以入手這堂超值課程，這絕對是累積個人作品和超高客製化的網站救星啊!!
而且也不用擔心沒有前端基礎，我自己也是完全不懂前端語法但仍然可以架設出一個看起來非常專業的網站，所以這堂課也很適合沒程式基礎的小白哦!

.
