前言

終於來到Python的重頭戲啦!說到Python最大的應用絕對非爬蟲莫屬，那爬蟲到底是什麼呢?
爬蟲簡單來說就是到提供數據的網站(ex:證交所)將想要的數據爬取下來，並在程式介面上進行數據分析。

.

但是由於過度頻繁地對網站進行request容易造成網站伺服器癱瘓，一般而言大多數的網站對於爬蟲都有一定程度的防爬蟲機制，像是短時間內限制request次數，或是直接使用驗證碼機制防止爬蟲。
而對於不同網站都會有不同的爬蟲技巧，本堂課程收錄了可以破解90%以上網頁的爬蟲技術，甚至可以爬到連網站上都無法顯示的data!

《一》認識網頁型態

差異 靜態網頁 動態網頁
程式技術 使用HTML、CSS做出的單獨網頁檔案 透過JavaScript非同步的方式去跟伺服器資料庫索取資料
優點 技術成分簡單，載入速度快 可因應不同需求展示不同內容，頁面管理有效率不需reload
缺點 無法因應不同需求展現不同內容 載入速度中等
範例 hsiu28.net Yahoo.com
Javascript Switcher 正常顯示 部分空白
後台 無 有
爬蟲難度 容易 困難

《三》證交所爬蟲

data型態為表格 ➨ 使用pd.read

import requests

# 將網址中的json改為csv ---refer to FinLab部落格
url = 'https://www.twse.com.tw/exchangeReport/MI_INDEX?response=csv&date=20210810&type=ALLBUT0999&_=1628643765268'
headers = {'User-Agent':'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/517.16 (KHTML, like Gecko) Chrome/12.0.4912.333 Safari/524.16'}

res = requests.get(url,headers=headers)

# 資料整理
import pandas as pd
import io

data = '\n'.join([l for l in res.text.split('\n') if len(l.split(',"'))>=12])

df = pd.read_csv(io.StringIO(data.replace('=',''))).rename(columns={'證券代號':'stock_id'}).set_index('stock_id')
df = df.applymap(lambda s:pd.to_numeric(str(s).replace(',',''),errors='coerce')).dropna(axis=1,how='all')

df

《四》公開資訊觀測站爬蟲

data型態為表格 ➨ 使用pd.read
查詢按鍵 ➨ 透過Form Data調整參數

import requests

url = 'https://mops.twse.com.tw/mops/web/ajax_t51sb01'
headers = {'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/888.36 (KHTML, like Gecko) Chrome/23.0.2313.192 Safari/123.34'}

# Form Data
payload = {
    'encodeURIComponent': '1',
    'step': '1',
    'firstin': '1',
    'TYPEK': 'sii' # 上市:sii / 上櫃:otc
}

res = requests.post(url,headers=headers,data=payload)

# 資料整理
import pandas as pd

df = pd.read_html(res.text)[0].rename(columns={'公司代號':'stock_id'}).set_index('stock_id')
df

《五》期交所爬蟲

下載csv檔 ➨ 使用pd.read
下載按鍵 ➨ 透過Form Data調整參數

import requests
import datetime

StartDate = datetime.date(2007,8,10)
EndDate = StartDate+datetime.timedelta(365*3)

url = 'https://www.taifex.com.tw/cht/3/futAndOptDateDown'
headers = {'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/333.33 (KHTML, like Gecko) Chrome/11.0.6665.188 Safari/999.36'}

# Form Data
payload = {
    'firstDate': f"{StartDate.strftime('%Y/%m/%d')}", # 最舊可索取data ~ 期間為三年
    'lastDate': f"{EndDate.strftime('%Y/%m/%d')}", # 當天日期
    'queryStartDate': f"{StartDate.strftime('%Y/%m/%d')}", # 索取起始日期
    'queryEndDate': f"{EndDate.strftime('%Y/%m/%d')}", # 索取結束日期
}

res = requests.post(url,headers=headers,data=payload)

# 資料整理
import pandas as pd
import io

df = pd.read_csv(io.StringIO(res.text)).rename(columns={'日期':'date'}).set_index(['date','身份別'])
df = df.applymap(lambda s:pd.to_numeric(str(s),errors='coerce'))

df

《六》蝦皮爬蟲

data型態為文字 ➨ 使用res.json() / beautifulsoup

import requests

url = 'https://shopee.tw/api/v4/search/search_items?by=relevancy&keyword=jordan&limit=60&newest=0&order=desc&page_type=search&scenario=PAGE_GLOBAL_SEARCH&version=2'
headers = {'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/111.26 (KHTML, like Gecko) Chrome/33.0.6785.145 Safari/666.11'}

res = requests.get(url,headers=headers)

# 資料整理
names = [product['item_basic']['name'] for product in res.json()['items']]
prices = [int(product['item_basic']['price'])/10e4 for product in res.json()['items']]

import pandas as pd

df = pd.DataFrame({
    'price':prices
},index=[names])

df

《八》CMoney爬蟲

防爬蟲機制 ➨ 使用selenuim
安裝chromedriver (查看本機版本:點選網頁自訂及管理 ➨ 說明 ➨ 關於Google Chrome)

from selenium import webdriver

driver = webdriver.Chrome('chromedriver.exe')
url = 'https://www.cmoney.tw/finance/f00039.aspx?s=2615'

driver.get(url)
driver.maximize_window() # 最大化視窗

# 資料整理
import pandas as pd

df = pd.DataFrame()

for i in range(2,40):
    try:
        df = df.append(pd.DataFrame({
            '買張':driver.find_elements_by_xpath(f"//*[@id='MainContent']/ul/li[1]/article/div/div/div/div/table/tbody/tr[{i}]")[0].text.split(' ')[1],
            '賣張':driver.find_elements_by_xpath(f"//*[@id='MainContent']/ul/li[1]/article/div/div/div/div/table/tbody/tr[{i}]")[0].text.split(' ')[2],
            '買賣超':driver.find_elements_by_xpath(f"//*[@id='MainContent']/ul/li[1]/article/div/div/div/div/table/tbody/tr[{i}]")[0].text.split(' ')[3],
        },index=[driver.find_elements_by_xpath(f"//*[@id='MainContent']/ul/li[1]/article/div/div/div/div/table/tbody/tr[{i}]")[0].text.split(' ')[0]]))
    except:
        for i in range(2,40):
            try:
                df = df.append(pd.DataFrame({
                    '買張':driver.find_elements_by_xpath(f"//*[@id='MainContent']/ul/li[2]/article/div/div/div/div/table/tbody/tr[{i}]")[0].text.split(' ')[1],
                    '賣張':driver.find_elements_by_xpath(f"//*[@id='MainContent']/ul/li[2]/article/div/div/div/div/table/tbody/tr[{i}]")[0].text.split(' ')[2],
                    '買賣超':driver.find_elements_by_xpath(f"//*[@id='MainContent']/ul/li[2]/article/div/div/div/div/table/tbody/tr[{i}]")[0].text.split(' ')[3],
                },index=[driver.find_elements_by_xpath(f"//*[@id='MainContent']/ul/li[2]/article/div/div/div/div/table/tbody/tr[{i}]")[0].text.split(' ')[0]]))
            except:
                pass
            pass
        break

dataset = df.applymap(lambda s:pd.to_numeric(str(s),errors='coerce'))
dataset = dataset.sort_values('買賣超',ascending=False)
dataset

結論

這堂課程提供了幾乎可以爬90%以上的爬蟲技巧，相信大家可以透過這些技術將自己想蒐集的資料給爬取下來進行分析，但大家一定要特別注意爬蟲本身就是有點遊走在法律及道德邊緣的方法，所以在爬蟲時也務必要遵守網路秩序，不要將爬蟲技巧用在惡意攻擊網站上，以免造成網站管理員的損害。
