《二》Machine Learning (機器學習)

1.認識機器學習
參數 意義
Feaures 訓練model從哪些角度進行學習
Label 正確解答
種類 特色 原理
Supervised (監督式學習) 有features & label Input features，Output label，訓練model搜尋出兩者之間的關係。
Unsupervised (非監督式學習) 只有features Input features，沒有Output，訓練model搜尋樣本間的規律關係。相似度比對 (clustering)
Reinforcements (強化學習) 有features & label(分數獎懲) 與環境互動獲得回饋
演算法 原理 Output
Classification label為不連續值(種類) ➨ 分類 True / False
Regression label為連續值(數值) ➨ 規律性 預測數值

結論

看完上述AI預測成果後就可以發現，透過演算法和類神經元的訓練後竟然可以打造出幾乎可以拿來實戰的AI投資策略，其實國內外已經陸續有少數基金機構將少數的資金拿來真正做AI交易。
但以現今的技術想要達成AI完全替代傳統人工投資還要一大段路要走，最大的問題出在模型的黑盒化，大部分的時候我們只會得到模型output的交易訊號，但卻沒辦法知道AI的思考方式，這對於要落實AI交易自動化來說是非常巨大的風險。
舉例來說，若某天市場出現類似2008金融風暴或是2020疫情融斷等黑天鵝事件時，我們無法得知AI會做出何種判斷，有可能做出將全數部位一次出清，而這種出清方式將會造成市場劇烈下跌，進一步擴大投資資金的損失。而到時責任歸屬也是一大問題。

.

即使AI在金融方面的應用還無法像其他領域這麼成功，但是未來的發展潛力絕對不可小覷，所以機器學習、神經網路也逐漸成為python科學家必備的基礎知識囉!
